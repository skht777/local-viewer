# プロジェクト概要

## サービス名
Local Content Viewer

## 目的
ローカルディレクトリの画像・動画・PDFを快適に閲覧するWebアプリケーション。
Dockerで配布・実行する個人用ビューワー。

## 主な機能
- ZIP/RAR/7z内の画像を直接表示（展開不要）
- MP4等の動画を埋め込みプレイヤーで再生
- PDFをページ単位で画像として表示
- CGモード（1枚ずつ表示）/ マンガモード（縦スクロール）
- 動画タブ（埋め込みフィード）
- トップページ（マウントポイント一覧）→ ファイルブラウザー → ビューワー
- キーボードナビゲーション（WASD + 矢印キー）
- ファイル名キーワード検索（SQLite FTS5）

## 基本フロー
1. Dockerコンテナ起動、ローカルディレクトリをマウント
2. トップページでマウントポイントを選択
3. ディレクトリツリー + サムネイルグリッドでブラウズ
4. 画像クリック → CGモード/マンガモードで閲覧
5. 動画はブラウザー画面内で直接再生
