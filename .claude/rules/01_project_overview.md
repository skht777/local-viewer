# プロジェクト概要

## サービス名
Local Content Viewer

## 目的
ローカルディレクトリの画像・動画・PDFを快適に閲覧するWebアプリケーション。
Dockerで配布・実行する個人用ビューワー。

## 主な機能
- ZIP/RAR/7z内の画像を直接表示（展開不要）
- MP4等の動画を埋め込みプレイヤーで再生（MKV は自動 MP4 変換）
- PDFをページ単位で画像として表示（テキスト選択対応）
- CGモード（1枚ずつ表示）/ マンガモード（縦スクロール連続表示）
- 見開き表示（通常 / 表紙単独）
- 動画タブ（埋め込みフィード、再生位置保存）
- ディレクトリ/アーカイブのプレビューサムネイル
- トップページ（マウントポイント一覧）→ ファイルブラウザー → ビューワー
- キーボードナビゲーション（WASD + 矢印キー）
- セット間ジャンプ（再帰探索 + バックグラウンドプリフェッチ）
- ファイル名キーワード検索（SQLite FTS5 trigram）
- `manage_mounts.sh` Bash TUI でマウントポイントを追加・削除・一覧管理（ホスト側で実行）
- `init.sh` で初回セットアップ（.env コピー + manage_mounts.sh）

## 基本フロー
1. `./init.sh` で初回セットアップ → `./start.sh` で Docker コンテナ起動
2. トップページでマウントポイントを選択（GET /api/mounts）
3. ディレクトリツリー + サムネイルグリッドでブラウズ
4. 画像クリック → CGモード/マンガモードで閲覧
5. 動画はブラウザー画面内で直接再生
